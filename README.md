# Mujeres Activistas Ambientales en Latinoamérica 
## Análisis de datos con Python y SQL

## Descripción
Análisis exploratorio del perfil, trayectoria, impacto y
situación de seguridad de 10 mujeres activistas ambientales
en Latinoamérica. Fuentes: Global Witness, Front Line
Defenders y AWID.

## Hallazgos principales
1. El 80% de las activistas han recibido amenazas.
   El reconocimiento internacional NO las protege —
   el 88% de las premiadas también fue amenazada.

2. Dedican en promedio 21.7 años a defender sus territorios.
   Tarcila Rivera lleva 43 años — la trayectoria más larga.

3. Entre las 10 han ganado 21 campañas ambientales.
   Nemonte Nenquimo es la más eficiente — 2 victorias
   en solo 8 años de activismo.

## Países analizados
Honduras · Colombia · Ecuador · Perú · México · Nicaragua

## Herramientas
- Python (Google Colab)
- pandas · matplotlib
- SQL (SQLiteOnline)
- Fuentes: Global Witness · Front Line Defenders · AWID

## Estructura del análisis
1. Carga y exploración del dataset
2. Análisis de seguridad y amenazas
3. Perfil y trayectoria de activistas
4. Impacto y reconocimiento internacional
5. Informe final con reflexión
6. Consultas SQL con CASE WHEN y GROUP BY
